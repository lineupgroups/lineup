## 🚀 **How to Use**

### **Add a Testimonial:**
1. Go to `/admin` → Landing Page → **Testimonials** tab
2. Click **"Add Testimonial"** button (orange)
3. Fill in the form:
   - Select type: Creator or Supporter
   - Enter name (required)
   - Enter quote (required, max 300 chars)
   - Fill optional fields (role, location, photo, amount)
   - Click star rating to set (1-5 stars)
   - Toggle "Featured" if you want it on landing page
   - Set display order (lower shows first)
   - Set status: Draft or Published
4. Click **"Save"**
5. ✅ Done!

### **Add an FAQ:**
1. Go to `/admin` → Landing Page → **FAQ** tab
2. Click **"Add FAQ"** button (orange)
3. Fill in the form:
   - Enter question (required)
   - Enter answer (required)
   - Select category
   - Toggle "Featured" if you want it on landing page
   - Set display order
   - Set status: Draft or Published
4. Click **"Save"**
5. ✅ Done!

### **Edit Existing Items:**
1. Find the item in the list
2. Click the blue **Edit** button (pencil icon)
3. Form opens with current values
4. Make changes
5. Click **"Save"**
6. ✅ Updated!

### **Delete Items:**
1. Find the item in the list
2. Click the red **Delete** button (trash icon)
3. Confirm the prompt
4. ✅ Deleted!
